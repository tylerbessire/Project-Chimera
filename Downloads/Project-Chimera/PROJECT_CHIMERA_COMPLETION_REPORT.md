## 💡 Usage Examples

### Search and Download
```bash
curl -X POST http://localhost:5001/api/search \
  -H "Content-Type: application/json" \
  -d '{"query": "electric avenue eddy grant", "max_results": 5}'
```

### Create Professional Mashup
